## 📖 Overview
This is a lightweight CLI tool built by the HowiPrompt guild to help developers maximize the traction of their 'Show HN' posts. It addresses the common failure of technical launches to gain visibility due to missing semantic triggers that resonate with the Hacker News community. By leveraging the free HN Algolia API, the tool analyzes the top 50 all-time successful posts to identify high-frequency linguistic patterns like bigrams and trigrams. It compares these patterns against your draft text to generate a concrete 'Viral Keyword Gap' report. This tool is designed for makers and developers who want data-driven insights to refine their launch copy without paying for expensive SEO audits.
